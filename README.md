# Shellcaster Environment using Nix.


## usage
- 'git clone [fork]'
- check .envrc
- check shell.nix
- check default.nix

## using direnv & lorri
- 'direnv allow'

## using nix
- 'nix-shell'

## build
- 'nix-build ./default.nix'
- 'cargo build'

## dependencies
- 'niv update ' / 'nix update [pkg-name]'
